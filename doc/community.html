boomerang community

boomerang is an opensource javascript library released under the BSD license. It depends on its community to keep it alive.

Source code

The source code is on github at github.com/yahoo/boomerang. Feel free to fork it and contribute to it. You can also get a tarball or zip archive of the code.

Bugs

Use github's issue tracking system to report or follow bugs.

Discussion

boomerang has its own discussion forum on the Yahoo! Developer Network website. Feel free to use it for all boomerang related discussions.
