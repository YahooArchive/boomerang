The boomerang API

core

  • BOOMR — The basic beaconing object.
  • BOOMR.utils — Utility functions within the BOOMR object.

core plugins

  • BOOMR.plugins.RT — The roundtrip plugin that measures page load time.
  • BOOMR.plugins.BW — The bandwidth plugin that measures connection bandwidth and latency.

extra plugins

  • BOOMR.plugins.DNS — The DNS latency plugin that measures latency of the user's DNS.
  • BOOMR.plugins.IPv6 — The IPv6 latency plugin that measures various IPv6 related metrics.
  • BOOMR.plugins.NavigationTiming — A plugin that collects performance data from user agents that implement the W3C Navigation Timing specification.

The latest code and docs is available on github.com/yahoo/boomerang
