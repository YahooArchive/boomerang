どのようにして boomerang を使うか

以下は ユースケース の中であげられている項目です。

使用方法

  • 使用方法 0 — ビーコンや before_beacon イベントハンドラーからデータを取得する
  • 使用方法 1a — ページ上のリンクをユーザーがクリックし onload イベントが発生した時にページは有効になる
  • 使用方法 1b — ページ上のリンクをユーザーがクリックし開発者が決めたポイントでページを有効にできる
  • 使用方法 2 — 動的な読み込みのパフォーマンスを認識して測定する
  • 使用方法 3 — ページの読み込み時間と一緒にユーザーの帯域幅や遅延を測定する
  • 使用方法 4 — さらに正確なページの読み込み時間を測定する
  • 使用方法 5 — リクエストとページにタグをつける
  • 使用方法 6 — boomerang を設定する
  • 使用方法 7 — 不正なビーコンを防ぐ
  • 使用方法 8 — DNS の遅延を測定する
  • 使用方法 9 — W3C Navigation Timing API からパフォーマンスデータを回収する
  • 使用方法 10 — Google Chrome による prerendered されたページの読み込み時間
