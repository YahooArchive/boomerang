boomerang コミュニティ

boomerang は BSD ライセンス のもとでリリースされたオープンソースの JavaScript ライブラリーです。コミュニティ上の活動もそれに依存します。

ソースコード

ソースコードは GitHub の github.com/yahoo/boomerang に公開されています。自由に fork して貢献してください。また、ソースコードの tarball や zip アーカイブ も取得できます。

バグ

バグの報告には GitHub の issue tracking system を利用します。

ディスカッション

boomerang は Yahoo! Developer Network 上に専用の ディスカッションフォーラム があります。boomerang に関するディスカッションに自由に利用してください。
