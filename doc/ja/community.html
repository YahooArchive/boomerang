boomerang コミュニティ

boomerang は BSD ライセンス のもとでリリースされたオープンソースの JavaScript ライブラリーです。コミュニティ上の活動もそれに依存します。

ソースコード

ソースコードは GitHub の github.com/lognormal/boomerang に公開されています。自由に fork して貢献してください。また、ソースコードの tarball や zip アーカイブ も取得できます。

バグとディスカッション

バグの報告や機能のディスカッションには GitHub の issue tracking system を利用します。
