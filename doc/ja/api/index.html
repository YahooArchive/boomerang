boomerang API

コア

  • BOOMR — ビーコン基本オブジェクト
  • BOOMR.utils — BOOMR オブジェクトのユーティリティ関数

コアプラグイン

  • BOOMR.plugins.RT — ページの読み込み時間を測定するプラグイン
  • BOOMR.plugins.BW — コネクションの帯域幅と遅延を測定するプラグイン

追加プラグイン

  • BOOMR.plugins.DNS — ユーザーの DNS の遅延を測定するプラグイン

最新のソースコードとドキュメントは github.com/yahoo/boomerang に公開されています。
