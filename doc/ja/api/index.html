ドキュメント一覧

boomerang API

コア

  • BOOMR — ビーコン基本オブジェクト
  • BOOMR.utils — BOOMR オブジェクトのユーティリティ関数

コアプラグイン

  • BOOMR.plugins.RT — ページの読み込み時間を測定するプラグイン
  • BOOMR.plugins.BW — コネクションの帯域幅と遅延を測定するプラグイン

追加プラグイン

  • BOOMR.plugins.DNS — ユーザーの DNS の遅延を測定するプラグイン
  • BOOMR.plugins.IPv6 — IPv6 関連のさまざまな測定を行う IPv6 レイテンシープラグイン
  • BOOMR.plugins.NavigationTiming — W3C Navigation Timing 仕様を実装しているユーザーエージェントからパフォーマンスデータを収集するプラグイン

最新のソースコードとドキュメントは github.com/lognormal/boomerang に公開されています。
